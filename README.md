- 코테 합격 기원 :trophy:



## Commit convention

* 기본 형태: `[태그] message`
* 태그 규칙
  * `[solve]`: 해결한 문제 코드
  * `[fail]`: 실패한 문제 코드
  * `[settings]`: git 설정 관련 수정(gitignore or README 수정 등)
  * `[learn]`: 알고리즘 이론 학습 내용
* message 규칙
  * 모든 message는 영어로 작성한다
  * 문제는 출처와 문제번호를 `BOJ_12345` 형태로 입력한다
